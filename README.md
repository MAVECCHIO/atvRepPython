# atvRepPython
Este programa interage com o usuário de duas maneiras:

Pergunta o nome e a idade do usuário, e informa se ele é maior ou menor de idade.

Solicita que o usuário insira um número e exibe a tabuada desse número.

Como Funciona:
1. Verificação de Idade:

O programa pede ao usuário seu nome e idade.

Com base na idade fornecida, ele informa se o usuário é maior ou menor de idade.

2. Exibição da Tabuada:

O programa solicita ao usuário que insira um número.

Com base no número fornecido, ele exibe a tabuada de 1 a 10 desse número.

Como Usar:

Execute o script.

O programa pedirá seu nome e idade.

Em seguida, o programa informará se você é maior de idade ou menor de idade.

O programa pedirá um número para exibir a tabuada desse número.

A tabuada será mostrada de 1 a 10.
